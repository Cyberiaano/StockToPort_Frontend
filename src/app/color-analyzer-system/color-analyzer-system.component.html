Color Analyzers

A device that captures images and compares color to references with smart technology. Applied to conveyors at warehouse exit and port for real-time monitoring of fertilizer quality.

Features

Color Detection

Detect specific color thresholds or variations indicative of contaminants.

Real-time Alerts

Provide real-time alerts when color variations exceed acceptable thresholds.

Data Logging

Record and log color analysis data over time for trend analysis.

Calibration

Calibrate the cameras to identify and adjust for color nuances in-house.

Impact/ Value Creation

  • Fewer dedicated quality inspectors needed.
  • Improved detection rate and accuracy.
  • Early detection decreases the impact of contaminations.
