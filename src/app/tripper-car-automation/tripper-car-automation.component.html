Tripper Car Automation

Automatic control of tripper cars using sensors communicating with the ICS (Integrated Control System). The solution provides remote control capabilities, load sensing technology, safety interlocks, emergency stop functions, local control panels, and manual override options to ensure safe and efficient operations.

Features

Remote Control Capability

Enables remote operation of tripper cars for improved performance and safety.

Load Sensing Technology

Integrated sensors measure the capacity and weight of materials accurately.

Safety Interlocks

Prevent unauthorized access and ensure safe operating conditions.

Emergency Stop Function

Immediate halting of operations in critical situations to prevent accidents.

Manual Override

Provides redundancy and manual control in unforeseen situations.

Impact/ Value Creation

  • Precise stockpile stacking and better utilization of storage space.
  • Reduced need for FTE surveillance through integration with CCTV systems.
